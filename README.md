# TUN 模块开启脚本

在OpenVZ机器上无需使用面板干预开启TUN模块

已集成至：https://github.com/Misaka-blog/Misaka-WARP-Script

## 使用方法

```script
wget -N https://raw.githubusercontents.com/Misaka-blog/tun-script/master/tun.sh && bash tun.sh
```
