# TUN 模块开启脚本

在LXC/OVZ机器上无需使用面板干预开启TUN模块

## 使用方法

```script
wget -N https://raw.githubusercontents.com/Misaka-blog/tun-script/master/tun.sh && bash tun.sh
```
